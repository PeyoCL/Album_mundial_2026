# Album Mundial 2026

Aplicacion web estatica para llevar el control de un album del Mundial 2026.

## Archivos principales

- `index.html`: estructura de la app.
- `style.css`: estilos visuales y responsivos.
- `app.js`: interaccion, filtros, progreso y almacenamiento local.
- `data.js`: datos de selecciones, secciones y laminas.

## Como abrirlo

Abre `index.html` en el navegador.

El progreso se guarda en el `localStorage` del navegador.
