# Álbum Mundial 2026 - Gestor de Láminas

Aplicación web progresiva (PWA) para gestionar tu colección de láminas del Mundial 2026. Construida en HTML, CSS y JavaScript puro.

## Características

- Control de láminas y repetidas.
- Filtros por equipo, grupo, texto y lámina.
- Ordenamiento dinámico (Más completos, Menos, A-Z).
- Estadísticas en tiempo real.
- Exportación de listas de faltantes y repetidas en PDF/Excel.
- Backup local e importación/exportación JSON.
- PWA Instalable (Funciona offline y directo desde el escritorio/teléfono móvil).
- Modo oscuro/claro y diseño responsive.

## Uso

1. Sube los archivos a un servicio de alojamiento que soporte HTTPS (ej. GitHub Pages, Vercel, Netlify).
2. Abre la URL en cualquier navegador web moderno.
3. El navegador te ofrecerá la opción de instalar la aplicación para verla como una App Nativa.
4. Comienza a marcar tus láminas directamente con un clic (clic de nuevo para marcar como repetidas).
